Tipos de listener

  • Raton
    • click: click
    • dbclick: doble click
    • mouseenter: mouse entra a un elemento
    • mouseleave: mouse sale del elemento
    • mousedown: se presiona boton del mouse mientras el puntero esta en el elemento
    • mouseup: cuando se suelta un botón del ratón después de hacer clic en un elemento.
    • mousemove: cuando se suelta un botón del ratón después de hacer clic en un elemento.
  • Teclado
    • keydown: cuando se mantiene presionada una tecla
    • keyup: cuando se deja de presionar
  • Formulario
    • submit: cuando se envia un formulario
    • input: cuando se cambia el valor del input
    • change: cuando cambia el valor de un input y se confirma saliento del input
    • focus: cuando se selecciona un input
    • blur: cuando se deselecciona un input
  • Ventana
    • load: cuando se ha cargado la pagina
    • resize: cuando se ha cambiado el tamaño de ventana
    • scroll: cuando se desplaza por el contenido
  • Tiempo
    • setTimeout: esperar x milisegundos para algo
    • setInterval: cada x milisegundos repetir
  • Drag
    • dragstart: cuando se inicia el arrastre de un elemento
    • dragend: cuando se completa el arrastre
    • dragenter: cuando se arrastra un elemento sobre otro
    • dragleave: cuando se arrastra un elemento fuera de otro
